FAQ's About Task 1,2 and the POE

1.How does the program work?
Solution - Once the progam is running, the console will prompt the user to input values for income and all the expenses and it will allow the user to select an option if they are renting or if they want to purchase a property.Once an option is selected,the program will calculate the available money left for either renting or buying a property

2.How is the home loan monthly repayments calculated?
Solution - The repayments is calculated using simple interest formula (A = P(1+i*n))

3.Can the program calculate for both renting and buying a porperty?
Solution - No the program can only choose 1 option at a time and calculate the option the user has inputed

4.What net.framework version was used to create the WPF GUI?
Net.Framework 4.8

5.How to add labels and textboxes?
There are two ways you can add labels and textboxes.The first is to use to hard code them in XAML file which is the design page for the application
The second way is the drag and drop method. This method is much easier than the first because it auto generates the code in the XAML file. You can access these items under the toolbox which is located on the top left hand side of the screen
